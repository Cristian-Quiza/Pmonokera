# Pmonokera
Arquitectura de microservicios desarrollada con Ruby on Rails, PostgreSQL y RabbitMQ. Junto a un frontend en Next.js. Implementa APIs REST, comunicación entre servicios vía HTTP, publicación y consumo de eventos con Bunny, manejo de consistencia eventual y pruebas unitarias e integración siguiendo buenas prácticas de diseño y código limpio.
