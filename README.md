# P-monokera
Arquitectura de microservicios orientada a eventos desarrollada con Ruby on Rails, PostgreSQL y RabbitMQ. Junto a un frontend en Next.js que consume las APIs, Implementa APIs REST, comunicación entre servicios vía HTTP, publicación y consumo de eventos con Bunny, manejo de consistencia eventual, siguiendo buenas prácticas de diseño y código limpio.
